# Restaurante dos Guarás

Nós, **Brenno Pacheco** e **Emanuelle Victoria**, nos aventuramos no mundo do desenvolvimento web para a criação do site do **Restaurante dos Guarás**.

Este projeto surgiu da ideia de unir nossos conhecimentos de front-end com o uso de *vibe coding* para acelerar e aprimorar o desenvolvimento. 

Gostaríamos de destacar que, apesar da utilização do *vibe coding* para auxiliar no processo, nós estudamos e aprimoramos significativamente nosso próprio entendimento sobre front-end durante a construção deste projeto. Este foi um desafio muito especial e de grande aprendizado, visto que nossas áreas de atuação originais são bem diferentes: o meu foco é na área de **Dados**, enquanto o da Emanuelle é em **Infraestrutura**.

O site oficial do projeto estará disponível para acesso em: **[RESTAURANTEDOSGUARAS.COM.BR](https://restaurantedosguaras.com.br)**

## Tecnologias Utilizadas no Projeto
- React
- Vite
- TypeScript
- Tailwind CSS e shadcn-ui

## Como Executar Localmente
Para rodar este projeto em sua máquina:

1. Instale as dependências:
   ```bash
   npm install
   ```
2. Inicie o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```
